Stuff on GitHub

In my spare time I put on GitHub some solutions I've crafted myself for common problems on front end development. Here are some of them...

classing

Modern browsers include the classList API that has a great performance, but in some projects you can't take advantage of that boost because you need to cover old browsers (I'm looking at you IE). So this prototypes a function on HTML elements new class functions and if classList is available the API is used, if it's not then fallback functions are triggered.

eventually

This script allows you to add and remove events using a new function that implements the attach and detach event function if addEventLIstener and removeEventListener are not available, with fallbacks for stopPropagation and preventDefault.
